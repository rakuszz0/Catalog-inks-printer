# Product Photos Directory

Tempatkan gambar produk Anda di sini! Contoh penamaan file:
- `epson-664-black.jpg`
- `hp-680-tri-color.png`
- `canon-pg-47.webp`

## Tips:
1. Gunakan format gambar yang dioptimasi (JPG, PNG, atau WEBP)
2. Ukuran ideal: 400x400 pixel (square)
3. Gunakan nama file yang jelas dan deskriptif
